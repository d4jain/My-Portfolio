DSA Learning APP

The DSA Learning App is my final-year project, designed as an interactive chat-based teaching assistant for Data Structures and Algorithms (DSA) problems. The application allows users to submit LeetCode problem links along with their doubts, receiving guided hints and insights powered by GPT to enhance problem-solving skills without direct solutions. Built using Java and Spring Boot, the project focuses on a clean and intuitive chat interface, seamless backend integration, and well-structured prompts to facilitate learning. This experience has deepened my expertise in backend development, API integration, and AI-driven learning solutions.

Tech Stacks:

•Java •SpringBoot •Rest •ChatGPT API
